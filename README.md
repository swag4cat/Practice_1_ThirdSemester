# MiniDBMS — простая документно-ориентированная NoSQL СУБД

**Тема:** Реализация минимальной версии документно-ориентированной СУБД (MiniDBMS)  
**Цель:** Получить практические навыки в реализации фундаментальных структур данных, сериализации, парсинге запросов и проектировании систем хранения данных. Изучить принципы работы индексов в базах данных.

---

## Возможности

- Собственная реализация **HashMap** с цепочками и динамическим расширением  
- Поддержка **CRUD-операций**:
  - `insert` — вставка документа  
  - `find` — поиск по условиям  
  - `delete` — удаление по фильтру  
  - `create_index` — создание индексов
- **JSON**-хранение коллекций и индексов  
- Логические операторы: `$and`, `$or`  
- Операторы сравнения: `$eq`, `$gt`, `$lt`, `$like`, `$in`  
- **B-Tree индекс** для числовых полей (ускоряет `$eq`, `$gt`, `$lt`, `$in`)  
- Персистентность данных между запусками
